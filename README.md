## Install

1. Clone the repo
   ```sh
   git clone https://github.com/Kepler868/Tesla-Shop-Clone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

---

## Run

First, run the development server:

```bash
npm run start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
